# SafeHer - Women's Safety Platform

A comprehensive 3D React application designed for women's safety with interactive emergency features, real-time location tracking, and AI-powered safety tools.

## Features

### Emergency Actions
- **Emergency SOS**: One-tap 911 calling
- **Live Location**: Real-time GPS sharing with guardians
- **Panic Alarm**: Full-screen emergency mode with siren
- **Alert Guardians**: Instant notification to trusted contacts
- **Video Recording**: Capture video evidence
- **Voice Recording**: Record audio evidence

### Safety Tools
- **Safe Route**: AI-powered safest path finder
- **Crowd Detection**: Real-time safety zone analysis
- **Fake Call**: Simulate incoming calls
- **Safe Words**: Emergency code system
- **Health Monitor**: Track vitals during emergencies
- **AI Guardian**: 24/7 automated monitoring

### 3D Visuals
- **Animated 3D Shield**: Rotating metallic shield with glowing core
- **Starfield Background**: 1500+ animated stars
- **Interactive Lighting**: Purple and pink dynamic lights
- **Floating Animations**: Smooth 3D transitions

### Real-time Monitoring
- **Live Clock**: Current time and date display
- **GPS Tracking**: Real-time location coordinates
- **Battery Status**: Device battery level monitoring
- **Network Status**: Connection health indicator
- **Protection Status**: Overall safety system status

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm build
```

## Technologies

- React 18
- Three.js + React Three Fiber
- Framer Motion
- Lucide React Icons
- CSS3 with Glassmorphism

## Emergency Disclaimer

This platform is designed to assist in emergencies but should not replace official emergency services. Always call 911 or your local emergency number first in life-threatening situations.
